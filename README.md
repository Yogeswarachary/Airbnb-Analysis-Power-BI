# Airbnb Analysis using Power BI

An interactive Power BI dashboard project that analyzes Airbnb listings to uncover pricing trends, availability patterns, room type preferences, host behavior, and location-based insights.

## Overview

This project explores Airbnb listing data and transforms it into an interactive dashboard for business analysis. The goal is to help users understand how pricing, room type, availability, reviews, and neighborhood-level patterns affect the Airbnb market.

The dashboard is designed to support data-driven decisions for hosts, travelers, and analysts by converting raw listing data into visual and actionable insights.

## Business Objective

The main objective of this project is to analyze Airbnb data and answer questions such as:

- Which neighborhoods have the highest and lowest average prices?
- Which room types are most popular?
- How does availability vary across locations?
- Which hosts or listings contribute most to the platform?
- What patterns can be observed in reviews, occupancy, and booking behavior?

## Data Preparation

Before building the dashboard, the dataset was cleaned and transformed using Power Query.

Key steps included:

- Removing duplicate records
- Handling missing or inconsistent values
- Formatting columns correctly
- Cleaning categorical fields
- Preparing the data for analysis and modeling
- Creating calculated columns and measures using DAX

## Dashboard Features

The Power BI dashboard provides insights into:

- Average price by location and room type
- Availability patterns across neighborhoods
- Room type distribution
- Host and listing activity
- Review and rating trends
- Location-based booking patterns
- Overall Airbnb market behavior

## Key Insights

Some of the major insights from the analysis include:

- Certain neighborhoods show significantly higher average prices than others.
- Entire home/apartment listings generally have higher prices compared to shared rooms.
- Availability varies across neighborhoods, indicating differences in demand and occupancy potential.
- Review counts and listing activity help identify stronger-performing areas.
- Interactive filters make it easier to compare pricing and room type trends across locations.

## Business Use Cases

This dashboard can be useful for:

- Airbnb hosts evaluating pricing strategies
- Travel analysts studying market demand
- Property managers comparing neighborhood performance
- Business users tracking location-based trends
- Decision-makers analyzing hospitality market behavior
